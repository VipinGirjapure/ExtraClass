Explain Var, Let and Const

    Var

  • Var is a functional as well as Global Scope
  • Var can be reinitialized and redeclared
  • var can be modified later

    Let

  • Let is a Block Scope
  • Let cannot be reinitialized and redeclared
  • Let can be modified later

    Const

  • Const is a Block Scope
  • Const cannot be reinitialized and redeclared
  • Const cannot be modified later

What is Hoisting?

Hoising is a phenemenon by which we can access vaiables and functions even before we initialize it.In case of var it is undefined but in case of let and const it throws a Reference Error.

What is Temporal Dead Zone?

It is the time between the hoisting and the initialization of variables. Even let and const are hoisted but they are in different space called Temporal Dead Zone.

What are closures?

Closure is a function bundled together with its lexical(physical) environment. 

        function outer(){
            var num = 5;
            function inner(){
                console.log(num)
            }
            return inner
        }

        outer()() //5
        
        //we see 5 because inner forms Closure with outer.
Here the function will try to access the variables in local scope,if they dont find it then it will search it in the outer scope and still if it fails to find it will search in gloabl scope and finally if it is not found then it will return undefined.

Strings

It is Collection of characters.

String Length:

Length in a string tells the length of the string ie. how many characters are present in the string

            I/p:
            var str="hello";
            console.log(str.length);

            O/p:
            5

charAt:

charAt returns the character of the specified index

            I/p:
            var str="hello"
            console.log(charAt(1));

            O/p:
            e

IndexOf:

indexOf is a method used to return the index of the character in a string.

            I/p:
            var str="hello";
            console.log(str.indexOf("e"));

            O/p:
            1

Split:

split method returns the array of substrings.

            I/p:
            var str="hello";
            var data=str.split("");
            console.log(data);

            O/p:
            data[0]: "h"
            data[1]: "e"
            data[2]: "l"
            data[3]: "l"
            data[4]: "o"

Reverse:

string reverse is uses to reverse the original string.

            I/p:
            var str="hello";
            var arr=str.split("");
            var revArr=arr.reverse();
            var revArr=revArr.join("");
            console.log(revArr);

            O/p:
            olleh

Join:

It will join array of individual characters to a single string

            I/p:
            var arr=["h","e","l","l","o"];
            var newArr=arr.join("");
            console.log(newArr);

            O/p:
            hello

Trim:

trim is a method to remove white spaces in a string.

            I/p:
            var str="hello  ";
            var newStr=str.trim();
            console.log(newStr.length);//5
            console.log(str.length); //7

Slice:

slice is method in string is used to extract the part of the string and it does not modify the original string.

\ 
            I/p:
            var str="hello";
            var newStr=str.slice(0,3);
            console.log(newStr);

            O/p:
            hel

splice:

Splice is a method used to add/remove array elements.It also modifies the original original.

            I/p:

            Add elemnts:
            var arr=["hello","hogwards","harry","potter"];
            arr.splice(3,0,"dumbledore");
            console.log(arr);

            O/p:
            ['hello', 'hogwards', 'harry', 'dumbledore', 'potter'];

            I/p:

            Remove elements:

            var arr=["hello","hogwards","harry","potter"];
            arr.splice(0,1);
            console.log(arr);          //  ['hogwards', 'harry', 'potter']

Concatenation:

concatenation is a method is used to add two string to make a single string.

            
                var str1="hello";
                var str2="world";
                var str3=str1.concat(" "+str2);
                console.log(str3); //hello world

Replace:

Replace is method which only replaces the first string in the original string.

                var str="hello world world";
                var str=str.replace("world","earth");
                console.log(str); //hello earth world

ReplaceAll:

replaceAll is a method which replaces all occurences of a string in the original string.

            
                var str="hello world world";
                var str=str.replace("world","earth");
                console.log(str); // hello earth earth

toLowerCase:

toLowercase in a string is a method which converts the uppercase string to lowercase string.

                I/p:
                var str="HeLLo";
                str=str.toLowerCase();
                console.log(str);  //hello

toUpperCase:

toUpperCase is a method which converts the lowercase string to uppercase string.


                var str="hello";
                str=str.toUpperCase();
                console.log(str); //HELLO
