Explain Var, Let and Const

    Var

  • Var is a functional as well as Global Scope
  • Var can be reinitialized and redeclared
  • var can be modified later

    Let

  • Let is a Block Scope
  • Let cannot be reinitialized and redeclared
  • Let can be modified later

    Const

  • Const is a Block Scope
  • Const cannot be reinitialized and redeclared
  • Const cannot be modified later

What is Hoisting?

Hoising is a phenemenon by which we can access vaiables and functions even before we initialize it.In case of var it is undefined but in case of let and const it throws a Reference Error.

What is Temporal Dead Zone?

It is the time between the hoisting and the initialization of variables. Even let and const are hoisted but they are in different space called Temporal Dead Zone.

Decision Making

  1. if statement
  2. if...else statement
  3. if...else if...else statement

1.if statement

Syntax 

if(condition){
      //body of if  }

The 'if' statement evaluates the condition inside the parenthesis ().

  • If the condition is evaluated to true, the code inside the body of 'if' is executed.
  • If the condition is evaluated to false, the code inside the body of 'if' is skipped.
 Example:
    // check if the number is positive

    const number = 2;
    
    // check if number is greater than 0
    if (number > 0) {
     // the body of the if statement
      console.log("The number is positive");
    }
    
    //"The number is positive"

2.if...else statement

Syntax

    if (condition) {
    // block of code if condition is true
} else {
   // block of code if condition is false
}

If..else statement evaluates the condition inside the paranthesis().

    if the condition is true

  • the code inside 'if' condition is executed.

    • and if the condition is false

  • the code inside the body of 'else' is executed

3.if...else..if statement

Syntax

    if (condition1) {
        // code block 1
    } else if (condition2){
        // code block 2
    } else {
        // code block 3
    }
  • if condition1 is true then 'code block 1' statement is executed.
  • if condition1 is false then 'code block 2' statement is executed.
  • if 'else if' condition is also false then 'code block 3' statement is executed .
    Example:
    // check if the number if positive, negative or zero
    
    const number =-2;

// check if number is greater than 0
if (number > 0) {
    console.log("The number is positive");
}
// check if number is 0
else if (number == 0) {
  console.log("The number is 0");
}
// if number is neither greater than 0, nor zero
else {
    console.log("The number is negative");
}

output:
//The number is negative

Switch Case

The 'switch' statement evaluates an expression and executes the corresponding body that matches the expression's result.

The syntax of the 'switch' statement is:

switch(variable/expression) {
    case value1:  
        // body of case 1
        break;

    case value2:  
        // body of case 2
        break;

    case valueN:
        // body of case N
        break;

    default:
        // body of default
}

    The switch statement evaluates a variable/expression inside parentheses ().

  • If the result of the expression is equal to value1, its body is executed.
  • If the result of the expression is equal to value2, its body is executed.
  • This process goes on. If there is no matching case, the default body executes.
  • If the break statement is encountered, the switch statement ends.

Accessing Elements By Tag Name

'getElementsByTagName()' is a dom method to allows you to select an element by its tag name.

Syntax:

let elements = document.getElementsByTagName(tagName);

Accessing Elements By Id

'getElementById()' is a dom method to allows you to select an element by its id.

Syntax:

let element = document.getElementById(id);

Accessing Elements By Class Name

'getElementByClassName()' is a dom method to allows you to select an element by its class name.

Syntax:

   1. let elements = document.getElementsByClassName(classNames)
   2. let elements = parentElement.getElementsByClassName(classNames)

Accessing Elements By querySelector

'querySelector()' targets first occurence of given tag.

Syntax:

       let elements = document.querySelector('div')
       console.log(elements)

       //this will returns first 'div' tag.

Accessing Elements By querySelectorAll

'querySelectorAll()' targets all occurence of given tag.

Syntax:

    let elements = document.querySelectorAll('div')
    console.log(elements)
    //this returns NodeList of all 'div tags.'

//AND

//you can give a index to target tag's index

let elements = document.querySelectorAll('div')[1]
console.log(elements)

//this return div tag on 1 index in the NodeList of div's.
